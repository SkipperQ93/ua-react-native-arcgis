# ua-react-native-arcgis

React Native Wrapper For ArcGIS

## Installation

```sh
npm install ua-react-native-arcgis
```

## Usage

```js
import { UaReactNativeArcgisView } from "ua-react-native-arcgis";

// ...

<UaReactNativeArcgisView color="tomato" />
```
